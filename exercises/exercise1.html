Imágenes y enlaces

Las imágenes se insertan con la etiqueta img, que tiene un atributo src con la URL de la imagen y un atributo alt con un texto alternativo

Las imágenes se pueden insertar desde un archivo local o desde una URL externa. Si se inserta desde un archivo local, el atributo src debe contener la ruta relativa al archivo. Si se inserta desde una URL externa, el atributo src debe contener la URL completa.

El atributo alt es un texto alternativo que se muestra si la imagen no se puede cargar. El texto alternativo también es importante para la accesibilidad, ya que las personas con discapacidad visual pueden usar lectores de pantalla que leen el texto alternativo en lugar de la imagen.

Si no se tiene una imagen propia, se puede usar una imagen de ejemplo. En Internet existen sitios web que ofrecen imágenes a mode de ejemplo

  • Unsplash
  • Lorem Picsum
  • Placehold
  • Pexels
