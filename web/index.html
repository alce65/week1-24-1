Clasificación de etiquetas

Las etiquetas de HTML se pueden clasificar en:

  1. Etiquetas de estructura
  2. Etiquetas de texto
  3. Etiquetas de formato

Etiquetas de estructura

Las etiquetas de estructura son las que se utilizan para definir la estructura de la página web. Algunas de las etiquetas de estructura más utilizadas son:

  • header
  • nav
  • section
  • article
  • aside
  • footer

Etiquetas de texto

Las etiquetas de texto son las que se utilizan para mostrar texto en la página web. Algunas de las etiquetas de texto más utilizadas son:

  • p
  • h1, h2, h3, h4, h5, h6
  • span
  • strong
  • em

Etiquetas de formato

Las etiquetas de formato son las que se utilizan para dar formato al texto en la página web. Algunas de las etiquetas de formato más utilizadas son:

  • strong
  • em
  • mark
  • del
  • ins
